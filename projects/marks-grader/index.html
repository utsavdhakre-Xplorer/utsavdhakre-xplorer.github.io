Student Marks Grader (Python)

This is a Python command-line project. Run the following code in your terminal:

def grade(marks):
    if marks >= 90:
        return 'A+'
    elif marks >= 80:
        return 'A'
    elif marks >= 70:
        return 'B'
    elif marks >= 60:
        return 'C'
    else:
        return 'F'

marks = int(input('Enter marks: '))
print('Grade:', grade(marks))
